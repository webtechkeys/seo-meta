Add your SEO meta tags to the page header for individual pages or posts.

== Description ==

SEO Meta Webtechkeys is a simple plugin to add SEO meta tags like title, description, keywords, author, and canonical link to individual pages or posts.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/seo-meta-webtechkeys` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Edit any post or page, and you will see a "SEO Meta Tags" meta box on the right-hand side.

== Frequently Asked Questions ==

= How do I add meta tags to a page or post? =

Edit any post or page, and fill in the fields in the "SEO Meta Tags" meta box on the right-hand side.
